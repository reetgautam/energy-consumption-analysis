# Energy Consumption Analysis

## Project Overview
This project analyzes energy consumption data to uncover patterns and trends, and predicts future energy usage. It also includes an interactive **Streamlit app** to visualize predictions.

## Dataset
- **Source:** `Energy_consumption.csv`  
- **Description:** Contains energy usage data (e.g., time, consumption in kWh)  
- **Preprocessed Data:** `cleaned_energy_consumption.csv`  
- **Test Data:** `test_data.csv`
  
## Project Structure
energy-consumption-analysis/
├── cleaning.ipynb # Data cleaning and preprocessing notebook
├── stremlit.py # Streamlit web app
├── energy_consumption.pkl # Trained ML model
├── encoder.pkl # Encoder for categorical features
├── predicted_energy.csv # Predicted energy consumption
├── tempCodeRunnerFile.py # Temporary/test code
├── README.md
